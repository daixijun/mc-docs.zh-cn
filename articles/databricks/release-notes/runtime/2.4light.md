# <a name="databricks-light-24"></a>Databricks Light 2.4

Databricks 于 2019 年 2 月发布了此映像。

Databricks Light 2.4 基于 Apache Spark 2.4。

有关详细信息，包括有关创建 Databricks Light 群集的说明，请参阅 [Databricks Light](../../runtime/light.md#light)。
